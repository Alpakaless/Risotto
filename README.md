# Risotto
A modern dotfiles manager for Linux — available as a desktop/mobile app and website

## About
Risotto is an Open Source dotfile manager for linux developed as an Undergraduate thesis project. It provides a modern and intuitive way to manage, edit, create and share system configuration files (dotfiles).

Risotto is available in two ways: a native desktop application built in Rust, and a web platform built with Vue.js and Laravel
